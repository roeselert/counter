#Simple Counter App
## Use cases
### Increment a counter by tapping on the touch screen
### Reset the counter by pressing the screen for 5 seconds

## Technology
### PWA Html, CSS and Plain JavaScript

## Design
### scandinavian design with light colors

## Development & deployment
### First plan the implementation and add a section to implementation.md
### Deploy the app at the end of a session by pushing to the git repo

## Acceptence test
# WHEN I open the app 
THAN the counter is set to zero 
AND the current value is shown
# WHEN I tap the screen 
THAN the counter is incremented by one 
AND the value is shown
# WHEN I tap the screen and hold the tap for at least 5 seconds
THAN the count is reset to 0
AND the value is shown on the screen
